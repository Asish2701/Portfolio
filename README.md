# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1kC2c7lvz11F4FQY9ChmOE0ZJWJXnpEg7

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (Optional) Set the `HUGGINGFACE_API_KEY` in [.env.local](.env.local) to enable the AI assistant. Get a free API key at https://huggingface.co/settings/tokens
3. Run the app:
   `npm run dev`
